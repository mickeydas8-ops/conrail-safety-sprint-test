BLANK SCREEN FIX

This is a safer single-file build. The CSS and JavaScript are embedded inside index.html so the app will not go blank if app.js or styles.css are missing on GitHub Pages.

Upload ONLY these items to the TOP LEVEL of your GitHub repository:

index.html
manifest.webmanifest
icons/

Delete or ignore old app.js and styles.css. They are no longer required for this fix.

After uploading:
1. Commit changes.
2. Wait 1-3 minutes.
3. Open the GitHub Pages link.
4. Press Ctrl + Shift + R in Edge.
5. If the installed Edge app still shows blank, uninstall it from edge://apps and reinstall from the GitHub Pages link.
